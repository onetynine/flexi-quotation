# Flexi Quotation — Smart Rental

Desktop app for generating Flexi Rental quotations for **LINEAR CHANNEL SDN BHD (Smart Rental)**.

## Download

Go to [**Releases**](../../releases/latest) and download `Flexi Quotation Setup x.x.x.exe`.

## Installation

1. Run the installer `.exe`
2. Follow the setup wizard
3. App installs to your user folder — **no admin rights required**
4. On first launch, PHP and the database are set up automatically

## Features

- Create and manage rental quotations (PDF export)
- Customer and plan management
- Auto-calculated rental fees, SST tax, deposit
- Quotation numbering: `SR{station}{YYMM}{seq}` (e.g. `SRMV260103`)
- Settings: change database location, backup & restore, reset

## Requirements

- Windows 10 / 11 (64-bit)
- No additional software required — PHP is bundled

## Updates

The app checks for updates automatically on launch. When a new version is available, it downloads in the background and prompts you to restart.
